# Zombie Zone 🧟

A browser-based zombie shooter built with Phaser 3.

## Setup

1. Download `phaser.min.js` from [phaser.io/download](https://phaser.io/download) and place it in the `libs/` folder
2. Push everything to GitHub
3. Enable GitHub Pages: Settings → Pages → Deploy from branch (main, root)
4. Your game will be live at `https://YOUR-USERNAME.github.io/zombie-zone`

## Controls

| Action | Keyboard | Mobile |
|---|---|---|
| Move | WASD | Left joystick |
| Look | Mouse | Right swipe |
| Shoot | Left click | Fire button |
| Switch weapon | Q | Swap button |
| Reload | R | Reload button |
| Jump | Space | Jump button |
| Pause | ESC | Pause button |
